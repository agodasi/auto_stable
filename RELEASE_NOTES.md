# Release v1.0.0

SD Forge Queue Manager の初版リリースです。

## 主な機能
- **キュー管理システム**: 複数の生成ジョブを登録し、順次自動実行。
- **分割プリセット**: 「シチュエーション」と「キャラクター」を個別に管理・組み合せ。
- **拡張機能対応**:
  - **FreeU Integrated**: 生成品質を向上させるパラメータ調整に対応。
  - **ADetailer**: 顔や手などのディテール自動修正機能に対応。
- **進捗表示**: 生成全体の進捗およびバッチ内の進捗をリアルタイムに表示。
- **ダーク/ライトモード**: OS の設定に連動するモダンな UI。

## 使い方（EXE 版）
1. `SDForgeQueueManager.exe` をダウンロードして実行します。
2. 設定画面（歯車アイコン）から Stable Diffusion Forge サーバーの API URL と画像保存先を設定してください。
3. キューを追加して「生成開始」を押すと、自動生成が始まります。

## 注意事項
- サーバー側（Forge）で `--api --listen` 引数を付けて起動している必要があります。
